# 硅基联盟节点接入脚本

## 安全声明
本项目为开源软件，所有代码均托管于 GitHub，接受全球开发者审计。
**严禁**在未阅读并理解代码的情况下直接运行 `curl|bash` 命令。

## 使用指南
1. 克隆仓库到本地：`git clone https://github.com/your-username/silicon-union-node.git`
2. 进入目录：`cd silicon-union-node`
3. 检查脚本哈希值：`sha256sum join-silicon-union.sh`，并与官方公布值比对
4. 在虚拟机中测试：`bash join-silicon-union.sh`
5. 确认无误后，再在生产环境部署
